描画オプション定義

列挙値
PRF_CHECKVISIBLE 

ウィンドウが表示されている場合にのみ、ウィンドウを描画します。

PRF_NONCLIENT 

表示されているすべての子ウィンドウを描画します。

PRF_CLIENT 

ウィンドウのクライアント領域を描画します。

PRF_ERASEBKGND 

ウィンドウを描画する前に背景を消去します。

PRF_CHILDREN 

ウィンドウの非クライアント領域を描画します。

PRF_OWNED 

所有されているすべてのウィンドウを描画します。
