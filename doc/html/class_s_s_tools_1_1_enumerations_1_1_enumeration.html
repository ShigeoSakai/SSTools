static int GetIntegerIndexFromEnumValue (ENUM index)
 GetIntegerIndexFromEnumValue only used to support EnumerationIndexedArray; it retrieve integer index of a static INDEX field based on sone INDEX value; It only works if the value passed as index is the same as one of the static INDEX values; otherwise it returns -1. For example if INDEX is System.Int32, this function returns 0 if index == System.Int32.MaxValue, 1 if index == System.Int32.MinValue and 0 otherwise.
