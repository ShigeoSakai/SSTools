string Name [get]
 Name of the static field representing enumeration item.
 
string AbbreviatedName [get]
 Abbreviated name of the static field representing enumeration item; abbreviation is defined by the AbbreviationAttribute
 
string DisplayName [get]
 Name of the item based on DisplayNameAttribute; the purpose of this member is to provide human-readable name for an item; it the attribute is not available or does not resolve the name, default value is used: DisplayName = Name
 
string Description [get]
 Description of the item based on DescriptionAttribute; it the attribute is not available or does not resolve the name, default value is used: Description = null
 
Cardinal Index [get]
 Index of a static field corresponding to present instance of EnumerationItem in the order it appears in the declaration
 
object Value [get]
 Value of a static field corresponding to present instance of EnumerationItem. The type of Value is always ENUM if ENUM is enumeration type, otherwise it can be of any type.
